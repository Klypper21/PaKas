│ → Si hay error: buscar en CHECKLIST_PRUEBAS_FORMULARIO.md Fase 12        │
│                                                                              │
│ ❓ "Guardar muestra error"                                                 │
│ → Abre F12 Console y copia el error exacto                                │
│ → Verifica que estás logueado como admin                                   │
│ → Referencia: Debugging en CHECKLIST_PRUEBAS_FORMULARIO.md               │
│                                                                              │
│ ❓ "El producto se gardó pero no veo variaciones en BD"                    │
│ → Ve a Supabase SQL Editor                                                │
│ → Ejecuta: SELECT * FROM product_variations                               │
│ → Verifica: que existen registros con parent_product_id del producto     │
│                                                                              │
│ ❓ "La tienda no muestra selectores de color/talla"                        │
│ → En tienda.html, abre F12 Console                                        │
│ → Escribe: supabase.from('product_variations').select('*').limit(1)      │
│ → Si retorna data: hay variaciones (problema en tienda.js)               │
│ → Si retorna 0: no hay variaciones (crear producto en admin)             │
│                                                                              │
│ ❓ "Necesito más ayuda"                                                    │
│ → Referencia completa: Fase 12 del CHECKLIST_PRUEBAS_FORMULARIO.md      │
│ → Todos los pasos de debugging documentados                               │
