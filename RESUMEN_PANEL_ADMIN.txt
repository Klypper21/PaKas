┌─ FUNCIONALIDADES ─────────────────────────────────────────────────────────┐
│                                                                              │
│  ✨ CREAR VARIACIÓN                                                         │
│     • Selecciona producto → Click "Nueva variación"                        │
│     • Completa formulario (SKU, Color, Talla, Precio, Stock)              │
│     • Aparece inmediatamente en la lista                                   │
│                                                                              │
│  ✨ LEER / LISTAR                                                           │
│     • Dropdown de productos cargado automáticamente                        │
│     • Selecciona uno → se cargan todas sus variaciones                     │
│     • Tarjetas muestran: SKU, Color, Talla, Stock, Precio, Fecha          │
│     • Stock con color: Verde si hay, Rojo si agotado                       │
│                                                                              │
│  ✨ EDITAR VARIACIÓN                                                        │
│     • Click en button "Editar" de la tarjeta                               │
│     • Modal abre con datos pre-llenados                                    │
│     • Editar campos (stock es lo más común)                                │
│     • Click "Guardar variación" → actualiza                                │
│                                                                              │
│  ✨ ELIMINAR VARIACIÓN                                                      │
│     • Click en button "Eliminar" de la tarjeta                             │
│     • Pide confirmación con nombre de SKU                                  │
│     • Elimina permanentemente (⚠️ no se puede deshacer)                   │
│                                                                              │
│  ✨ VALIDACIONES                                                            │
│     • Campos requeridos: SKU, Color, Talla, Precio, Stock                  │
│     • Precio y Stock no pueden ser negativos                               │
│     • Mensajes de error/éxito en toast notifications                       │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ FLUJO DE DATOS ──────────────────────────────────────────────────────────┐
│                                                                              │
│  Admin abre panel                                                           │
│    ↓                                                                        │
│  admin-variations.js se initialize                                         │
│    ↓                                                                        │
│  SELECT products → Populate dropdown                                       │
│    ↓                                                                        │
│  Admin selecciona un producto                                              │
│    ↓                                                                        │
│  SELECT product_variations WHERE parent_product_id = ?                     │
│    ↓                                                                        │
│  Renderiza tarjetas con data                                               │
│    ↓                                                                        │
│  Admin elige acción (crear/editar/eliminar)                                │
│    ↓                                                                        │
│  Ejecuta INSERT/UPDATE/DELETE en BD                                        │
│    ↓                                                                        │
│  Recarga lista automáticamente                                             │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ INTEGRACIÓN CON TIENDA ──────────────────────────────────────────────────┐
│                                                                              │
│  Cuando creo variaciones en Admin:                                         │
│                                                                              │
│  product_variations table ← INSERT                                         │
│            ↓                                                                │
│  Cliente abre tienda.html → Ve producto                                   │
│            ↓                                                                │
│  Click en producto → Modal con selectores                                  │
│            ↓                                                                │
│  Variations.js carga datos de BD                                           │
│            ↓                                                                │
│  VariationsUI popula selectores dinámicamente                              │
│            ↓                                                                │
│  Cliente selecciona color+talla                                            │
│            ↓                                                                │
│  Stock y precio actualizan según variación                                 │
│            ↓                                                                │
│  Click "Agregar al carrito" → Guarda SKU                                   │
│            ↓                                                                │
│  Carrito muestra qué variación exacta compró                                │
